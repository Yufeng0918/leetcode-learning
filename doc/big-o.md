# Big(O)分析

| 算法                         | 递归关系             | 运行时间 |
| ---------------------------- | -------------------- | -------- |
| Binary search                | T(n)=T(n/2)+O(1)     | O(logn)  |
| Binary tree traversal        | T(n)=2T(n/2)+O(1)    | O(n)     |
| Optimal sorted matrix search | T(n)=2T(n/2)+O(logn) | O(n)     |
| Merge sort                   | T(n)=2T(n/2)+O(n)    | O(nlogn) |
